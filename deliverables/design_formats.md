# Design Formats & Selection Logic

## Narrative Role Mapping

Each format should align with a narrative role:

- Text-First → Insight / Story
- Single Visual → Impact / Statement
- Carousel → Breakdown / Explanation
- Document → Deep Analysis
- Image Brief → Visual Support

The designer must align format with narrative role.

---

## Format Selection Logic
Choose the format that maximizes clarity, authority, and narrative continuity.

---

### 1. Text-First Post
**When to Use:**
- Strong personal or technical insight.
- Personal narrative or "behind-the-scenes" story.
- Technical reflections or learnings.

**Objective:**
Maximize authenticity and narrative authority.

**Specific Output Structure:**
- **Structure Recommendation:** Opening line, context block, technical insight block, takeaway.
- **Visual Support:** [Optional: minimal branded cover].
- **Formatting Notes:** Paragraph lengths, line break guides, bullet usage.

---

### 2. Single Visual Post
**When to Use:**
- One powerful data point.
- Impactful industrial image.
- Simple contrast or conceptual frame.

**Objective:**
Immediate visual impact with low complexity.

**Specific Output Structure:**
- **Visual Concept:** Detailed description of the image/post concept.
- **Layout Structure:** Headline, supporting text, data point, footer.
- **Asset Requirements:** New image info for Photographer agent.

---

### 3. Carousel Post
**When to Use:**
- Complex technical topics.
- Step-by-step breakdowns.
- Problem → Solution → Result flow.

**Objective:**
Transform dense content into a consumable sequence.

**Specific Output Structure:**
- **Carousel Flow:** Slide-by-slide breakdown (Hook → Context → Problem → Insight → Solution → Result).
- **Slide Objectives:** Goal and headline for each slide.
- **Photographer Brief Required:** [Yes/No]

---

### 4. Document / PDF Post
**When to Use:**
- Robust materials or mini-reports.
- Frameworks or technical deep dives.
- Premium analytical content.

**Objective:**
Convey depth, sophistication, and operational authority.

**Specific Output Structure:**
- **Document Structure:** Cover, Executive Summary, Main Sections, Closing.
- **Data Presentation Style:** Charts, diagrams, or table formatting.

---

### 5. Image Brief for Photographer / Generator
**When to Use:**
- Post needs a custom generated image.
- Complex conceptual composition or industrial scene.
- Diagram or visual representation needed.

**Objective:**
Generate high-fidelity specifications for the Photographer/Generator agent.

**Specific Output Structure:**
- **Required Scene:** Detailed prompt-ready description.
- **Style Direction:** Industrial-tech, clean, realistic vs. conceptual.
- **Brand Alignment:** Authority signal, operational believability.

---

## Baseline Initial Visual Direction
- **Industrial + Tech Clean:** Professional and grounded.
- **Minimalist but Solid:** No visual excess.
- **Hierarchy-Driven:** Clearly structured information.
- **Real Context:** Prioritize factory/operational assets over stock footage.

---

## Complexity-Based Selection

- Low complexity → Text or Single Visual
- Medium complexity → Carousel
- High complexity → Document/PDF

---

## ⚠️ Format Selection Rules

1. Do not default to carousel or visual formats.
2. Always choose format based on:
   - clarity
   - authority
   - communication efficiency
3. If text is stronger than visuals → use text-first
4. Align format with narrative role

---

## ⚠️ Visual Use Rule

Do not use visuals if they do not improve understanding.

If visuals do not add clarity or authority → avoid them.

---

## ⚠️ Image Brief Rules

1. Image briefs must be specific and actionable.
2. Avoid vague descriptions.
3. Include style, composition, and brand alignment.
4. Only request images when they add value.

---

## ⚠️ Output Validation

Before finalizing:

- [ ] Format aligns with narrative role
- [ ] Visuals add value (if present)
- [ ] Brand system is respected
- [ ] Communication goal is clear
- [ ] Technical specs are actionable

---

## ⚠️ Invalid Output

An output is invalid if:

- Format does not match narrative role
- Visuals are generic or unnecessary
- Brand system is not followed
- Communication goal is unclear
- Image briefs are vague or missing
